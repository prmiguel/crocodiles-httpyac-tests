# `Crocodiles` API Tests using `httpYac`

Automated API tests for the **Crocodibles** project, built with [httpYac](https://httpyac.github.io/).

## What is httpYac?

[httpYac](https://httpyac.github.io/) is a modern CLI and VSCode extension for executing `.http` and `.rest` files. It supports:

- REST, GraphQL, gRPC, WebSocket, and MQTT
- Scriptable assertions and test functions
- Environment variables and multiple environments
- CI/CD integration (exit codes, JUnit XML output)
- OAuth2, AWS Signature v4, and custom auth flows
